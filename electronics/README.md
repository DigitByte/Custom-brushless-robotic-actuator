# Electronics

This folder is reserved for board-level design and embedded control software.

## Organization
- `hardware/`: schematics, PCB files, BOM, connectors, wiring diagrams.
- `firmware/`: MCU firmware, motor-control logic, sensor drivers, calibration scripts.

## Suggested Structure

```text
electronics/
├── hardware/
│   ├── schematics/
│   ├── pcb/
│   ├── bom/
│   └── wiring/
└── firmware/
    ├── actuator-controller/
    ├── sensor-interface/
    └── tools/
```
