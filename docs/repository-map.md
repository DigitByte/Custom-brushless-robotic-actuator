# Repository Map

## Mechanical
- `mechanical/cad/actuator/`: actuator CAD (STEP/STL and component files).
- `mechanical/cad/planetary-gearbox/`: planetary gearbox printable geometry.

## Electronics
- `electronics/hardware/`: schematics, PCB layouts, BOM, wiring docs.
- `electronics/firmware/`: embedded code for motor controller and sensors.

## Software
- `software/ros2/odrive-stack/ws_odrive/`: ROS2 workspace for ODrive hardware interface and control packages.

## Docs
- `docs/overview.md`: existing long-form project overview.
- `docs/repository-map.md`: this quick file map.
