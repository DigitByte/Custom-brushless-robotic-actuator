# Mechanical

This folder contains all actuator and gearbox mechanical design assets.

## Organization
- `cad/actuator/`: integrated actuator CAD files.
- `cad/planetary-gearbox/`: planetary gearbox parts.

## Recommended Additions
- `cad/source/`: native CAD source files (Fusion/SolidWorks/Onshape exports).
- `manufacturing/`: print settings, machining notes, tolerances, QC checklists.
